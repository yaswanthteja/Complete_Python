
- First install python

- Now we need to create a account on imagekit and get the public key and private key and url endpoint

- [Imagekit](https://imagekit.io/registration/)

- sign up and login here 
- go to developer aptions and get the public key and private key and url endpoint and save them in a .env file

- now create a folder named "app" and inside that create a file named "app.py" and copy the code from the basics section and paste it here

- create a project using the uv command

```bash 
uv init .
```

- now we need to install the required packages for our project

```bash
uv add -r requirements.txt
```
- now we need to run the application using the uv command

```bash
uv run  .\main.py
```
