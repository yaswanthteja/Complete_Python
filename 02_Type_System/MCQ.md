### Operators MCQs

1. **Which operator is used for exponentiation in Python?**
    - A) `**`
    - B) `*`
    - C) `^`
    - D) `//`
   
   **Answer:** A) `**`

2. **What does the following expression evaluate to? `10 % 3`**
    - A) 1
    - B) 3
    - C) 0
    - D) 10
   
   **Answer:** A) 1

3. **Which operator is used to check if two values are not equal in Python?**
    - A) `==`
    - B) `!=`
    - C) `<>`
    - D) `is not`
   
   **Answer:** B) `!=`

4. **What is the result of `10 // 3`?**
    - A) 3.33
    - B) 3
    - C) 4
    - D) 1
   
   **Answer:** B) 3

5. **Which operator has the highest precedence in Python?**
    - A) `+`
    - B) `*`
    - C) `**`
    - D) `//`
   
   **Answer:** C) `**`

6. **What does the expression `2 + 3 * 4` evaluate to?**
    - A) 20
    - B) 14
    - C) 12
    - D) 9
   
   **Answer:** B) 14

7. **What is the result of `15 % 4`?**
    - A) 1
    - B) 2
    - C) 3
    - D) 4
   
   **Answer:** C) 3

8. **Which operator is used for floor division in Python?**
    - A) `/`
    - B) `//`
    - C) `%`
    - D) `**`
   
   **Answer:** B) `//`

9. **What does the expression `5 ** 2` evaluate to?**
    - A) 25
    - B) 10
    - C) 20
    - D) 5
   
   **Answer:** A) 25

10. **Which operator can be used to concatenate two strings?**
    - A) `+`
    - B) `*`
    - C) `&`
    - D) `|`
   
   **Answer:** A) `+`

### Strings MCQs

1. **What will be the output of the following code? `'Hello' + ' World!'`**
    - A) `HelloWorld!`
    - B) `Hello World!`
    - C) `HelloWorld`
    - D) `Hello World`
   
   **Answer:** B) `Hello World!`

2. **Which method can be used to convert all characters in a string to lowercase?**
    - A) `lower()`
    - B) `upper()`
    - C) `capitalize()`
    - D) `casefold()`
   
   **Answer:** A) `lower()`

3. **What will be the output of the following code? `len('Python')`**
    - A) 5
    - B) 6
    - C) 7
    - D) 8
   
   **Answer:** B) 6

4. **Which of the following methods can be used to remove any whitespace from both the beginning and the end of a string?**
    - A) `strip()`
    - B) `trim()`
    - C) `cut()`
    - D) `slice()`
   
   **Answer:** A) `strip()`

5. **What will be the output of the following code? `print('Hello'.upper())`**
    - A) `hello`
    - B) `HELLO`
    - C) `Hello`
    - D) `hellO`
   
   **Answer:** B) `HELLO`

6. **What does the `split()` method do in Python?**
    - A) Concatenates strings
    - B) Splits a string into a list
    - C) Removes characters from a string
    - D) Reverses a string
   
   **Answer:** B) Splits a string into a list

7. **Which method can be used to find the index of the first occurrence of a substring in a string?**
    - A) `find()`
    - B) `index()`
    - C) `locate()`
    - D) `search()`
   
   **Answer:** A) `find()`

8. **What will be the result of the following code? `'abc'.replace('a', 'A')`**
    - A) `ABC`
    - B) `abc`
    - C) `Abc`
    - D) `aaabbbccc`
   
   **Answer:** C) `Abc`

9. **Which of the following methods returns the number of times a specified value occurs in a string?**
    - A) `count()`
    - B) `frequency()`
    - C) `len()`
    - D) `index()`
   
   **Answer:** A) `count()`

10. **What will be the output of the following code? `print('Hello, World!'.startswith('Hello'))`**
    - A) `True`
    - B) `False`
    - C) `None`
    - D) `Error`
   
   **Answer:** A) `True`

### Variables MCQs

1. **Which of the following is a valid variable name in Python?**

   **Answer:** C) `15`

3. **Which keyword is used to declare a variable in Python?**
